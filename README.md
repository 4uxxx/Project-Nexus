# OGFN:Project Nexus

**フォートナイト チャプター2 シーズン8**の世界をもう一度。
アジアサーバーで24時間安定稼働中。いつでもあの頃の島へ飛び込もう。

## 🚀 プロジェクト概要

**OGFN**は、チャプター2 シーズン8をいつでもプレイできるように再現したプライベートサーバープロジェクトです。アジア地域に最適化された低遅延環境で、快適なゲームプレイを提供します。
